# **App Name**: Invoice Management System

## Core Features:

- Invoice Management: Securely upload, manually enter, and track invoice statuses with automated calculations for overdue amounts.
- Automated Reminder System: Send email and WhatsApp reminders at customizable intervals, triggered by a daily cron job with delivery status tracking.
- Dashboard & Analytics: Visualize total outstanding debt, overdue invoice breakdowns, payment trends, and recent activity through exportable reports. Use the reasoning 'tool' to ensure appropriate visualization type is used.
- User Management & Access Control: Implement role-based access for Admin and Finance Team members with secure authentication and audit logging.
- Cloud Storage and Backup: Ensure real-time data synchronization, automatic backups, and offline capabilities with local data caching.
- Secure Data Transmission: Protect sensitive data with HTTPS encryption for all data transmission and PDF document encryption at rest in Storage.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF), evoking trust and efficiency.
- Background color: Light gray (#E0E0E0), for a clean, professional look.
- Accent color: Sea green (#2E8B57), for calls to action and key metrics.
- Body and headline font: 'Inter', a grotesque-style sans-serif, which will lend a modern and objective look.
- Use Material Design icons to maintain consistency with modern UI standards.
- Employ a responsive design with a sidebar menu for intuitive navigation.
- Subtle loading indicators and transitions to enhance user experience.
